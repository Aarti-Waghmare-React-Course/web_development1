## üîç Introduction to the JavaScript `replaceAll()` Method

The **`replaceAll()`** method returns a **new string** where **all matches** of a pattern (string or regex) are replaced by a new substring.

**Syntax:**

‚ö†Ô∏è When using a regex, it must include the global (g) flag, otherwise replaceAll() will throw an error.

match ‚Üí The matched substring.

üß© Like replace(), the replaceAll() method does not modify the original string.

‚öôÔ∏è How It Works
The regular expression /js/gi matches JS, Js, and js (case-insensitive).

For each match, the callback determines the proper replacement case:

"JS" ‚Üí "JavaScript"

"Js" ‚Üí "Javascript"

"js" ‚Üí "javascript"
