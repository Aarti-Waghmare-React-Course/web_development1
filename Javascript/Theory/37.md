# JavaScript Array Methods

This chapter provides all important JavaScript Array methods that allow you to manipulate, search, transform, and manage arrays effectively.

---

## **Section 1. Array Properties**

### **1. `length`**
Learn how to use the `length` property effectively to get or set the size of an array.

---

## **Section 2. Adding / Removing Elements**

### **1. `push()`**
Add one or more elements to the **end** of an array.

### **2. `unshift()`**
Add one or more elements to the **beginning** of an array.

### **3. `pop()`**
Remove the last element of an array.

### **4. `shift()`**
Remove the first element of an array.

### **5. `splice()`**
Insert, delete, or replace elements.

### **6. `slice()`**
Copy selected elements from an array (non-destructive).

---

## **Section 3. Finding Elements**

### **1. `indexOf()`**
Return the **first index** of the matching element.

### **2. `lastIndexOf()`**
Return the **last index** of the matching element.

### **3. `includes()`**
Check if an element exists in an array.

### **4. `find()`**
Find and return the **first element** that matches a condition.

### **5. `findIndex()`**
Find the **index** of the first element that matches a condition.

---

## **Section 4. High-Order Methods**

### **1. `map()`**
Transform every element of an array and return a new array.

### **2. `filter()`**
Filter elements based on a condition.

### **3. `reduce()`**
Reduce all elements to a single value (left → right).

### **4. `reduceRight()`**
Reduce all elements to a single value (right → left).

### **5. `every()`**
Check if **every** element passes a test.

### **6. `some()`**
Check if **at least one** element passes a test.

### **7. `sort()`**
Sort the elements of an array.

### **8. `forEach()`**
Loop through array elements (no return value).

### **9. `with()`**
Return a new array with an element replaced at a specific index.

---

## **Section 5. Manipulating Arrays**

### **1. `concat()`**
Merge two or more arrays into a new array.

---

## **Section 6. Creating Arrays**

### **1. `of()`**
Create an array from arguments.

### **2. `from()`**
Create arrays from iterable or array-like objects.

---

## **Section 7. Flattening Arrays**

### **1. `flat()`**
Flatten nested arrays up to a specific depth.

### **2. `flatMap()`**
Apply a mapping function and flatten the result.

---

## **Section 8. Arrays to Strings**

### **1. `join()`**
Convert all elements into a single string with a separator.

---

## **Section 9. Advanced Operations**

### **1. Array Destructuring**
Assign array values to separate variables.

### **2. Spread Operator (`...`)**
Expand elements of an array in expressions or function arguments.

---

## **Section 10. Accessing Elements**

### **1. `at()`**
Access elements using positive or negative indexes.

---

## **Section 11. Reversing Elements**

### **1. `reverse()`**
Reverse array elements **in-place**.

### **2. `toReversed()`**
Return a **new reversed array** without modifying the original.

---

## **Section 12. Multidimensional Arrays**

Learn how to create, access, and manipulate **arrays inside arrays**.
