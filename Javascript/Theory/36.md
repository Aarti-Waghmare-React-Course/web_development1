## Introduction to JavaScript Arrays

In JavaScript, an **array** is an ordered list of values. Each value is called an **element** and is stored at a numeric **index**.

### Key Characteristics:
- An array can store **mixed data types** (number, string, boolean, null, etc.).
- Arrays are **dynamic**—they grow/shrink automatically.

---

## Creating JavaScript Arrays

### 1. Using the `Array` constructor:

let scores = new Array();      // empty array
let scores2 = Array(10);       // array with size 10
let scores3 = new Array(9, 10, 8, 7, 6); // array with elements
let signs = new Array('Red');  // array with one element
You can omit new:

let artists = Array();

Accessing Array Elements
JavaScript arrays are zero-based indexed.

let mountains = ['Everest', 'Fuji', 'Nanga Parbat'];

console.log(mountains[0]); // Everest
console.log(mountains[1]); // Fuji
console.log(mountains[2]); // Nanga Parbat

Updating an element:
let mountains = ['Everest', 'Fuji', 'Nanga Parbat'];
mountains[2] = 'K2';

console.log(mountains); // [ 'Everest', 'Fuji', 'K2' ]

Getting Array Size
let mountains = ['Everest', 'Fuji', 'Nanga Parbat'];
console.log(mountains.length); // 3


Basic Array Operations

1) Append to the end → push()
let seas = ['Black Sea', 'Caribbean Sea', 'North Sea', 'Baltic Sea'];
seas.push('Red Sea');

console.log(seas);
Output:
[ 'Black Sea', 'Caribbean Sea', 'North Sea', 'Baltic Sea', 'Red Sea' ]

2) Add to the beginning → unshift()
let seas = ['Black Sea', 'Caribbean Sea', 'North Sea', 'Baltic Sea'];
seas.unshift('Red Sea');

console.log(seas);
Output:
[ 'Red Sea', 'Black Sea', 'Caribbean Sea', 'North Sea', 'Baltic Sea' ]

3) Remove from the end → pop()
let seas = ['Black Sea', 'Caribbean Sea', 'North Sea', 'Baltic Sea'];
const lastElement = seas.pop();
console.log(lastElement); 

Output:
Baltic Sea

4) Remove from the beginning → shift()
let seas = ['Black Sea', 'Caribbean Sea', 'North Sea', 'Baltic Sea'];
const firstElement = seas.shift();
console.log(firstElement);

Output:
Black Sea

5) Find index of an element → indexOf()
   
let seas = ['Black Sea', 'Caribbean Sea', 'North Sea', 'Baltic Sea'];
let index = seas.indexOf('North Sea');
console.log(index); // 2

6) Check if a value is an array → Array.isArray()
console.log(Array.isArray(seas)); // true


Summary
A JavaScript array is an ordered list of values.

Arrays can hold mixed types.

Arrays are dynamic (auto-growing or shrinking).

You can perform basic operations such as:
push(), unshift(), pop(), shift()
indexOf()
Array.isArray()
