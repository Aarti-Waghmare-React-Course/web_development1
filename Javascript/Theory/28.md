ðŸ”¹ Variable and Expression Substitution (Interpolation)

Template literals allow variables and expressions inside ${} blocks.

Example 1: Variable substitution
let firstName = 'John';
let lastName = 'Doe';

let greeting = `Hi ${firstName}, ${lastName}`;
console.log(greeting); // Hi John, Doe


Example 2: Expression substitution
let price = 8.99;
let tax = 0.1;

let netPrice = `Net Price: $${(price * (1 + tax)).toFixed(2)}`;
console.log(netPrice); // Net Price: $9.89


ðŸ”¹ Tagged Templates

Tagged templates allow you to process template literals with a custom function before returning a string.

Syntax:
tag`Hello ${name}`;

Here, tag is a function that receives the literal parts and substitution values.


Example:
function format(literals, ...substitutions) {
  let result = '';

  for (let i = 0; i < substitutions.length; i++) {
    result += literals[i];
    result += substitutions[i];
  }

  result += literals[literals.length - 1];
  return result;
}

let quantity = 9;
let priceEach = 8.99;

let output = format`${quantity} items cost $${(quantity * priceEach).toFixed(2)}.`;
console.log(output);
